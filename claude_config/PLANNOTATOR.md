# Plannotator

**Official docs**: https://plannotator.ai
**GitHub**: https://github.com/backnotprop/plannotator

Visual plan review and code annotation for AI coding agents. Browser-based, runs locally.

## Slash Commands

```bash
/plannotator-review     # Interactive code review with visual diff
/plannotator-last       # Annotate the last rendered plan
/plannotator-annotate   # Open interactive annotation UI
/plannotator-archive    # Browse saved plan decisions
```

Installed as a Claude Code plugin — invoke via slash commands, not the binary.
